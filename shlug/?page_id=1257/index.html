GPLHOST.COM

/* 赞助SHLUG.org网站VPS */

GPLHost 有两个目标。其一是发布高安全性、高可用的虚拟主机产品。 我们坚持以最低的市场价格，提供给您最优质的服务。在其它公司只有靠降低服务标准才能做到如此低价，我们是如何做到这点的呢？这就是我们的第二个目标。

GPLHost 一直致力于开放源代码的虚拟主机工具。在多数情况下，大公司的产品总是不开放源代码的，然后就很少有创新。

只有不断的面对各种各样的问题，我们才能更好的了解需求。如今我们已经解决了所有的问题，而且找到了一种通用的基于开源软件的解决方案。在这个信 条的指引 下，我们拥有了我们能够享用的第一款软件–我们的控制面板 (DTC)。这款软件的主要目的是让每个人建站的时候都能够方便的使用开源软件，而不必费心的去了解各种复杂的守护进程的配置文件。目前一些主机提供商已 经开始使用DTC(或者自己订制的版本)。 这使得他们得以降低成本，最终得以降低最终用户的收费标准(许多主机提供商使用昂贵的商业软件)。如果您正在寻找一款简单易用而又同样功能强大的软件来管 理您的独立主机，那么这款软件对您来说是最好不过的了。 DTC 能够在一台单独的服务器上管理成千上万的各种帐号(包括ftp、mail等账户)，而且，它支持常用的所有浏览器。

在不久的将来，更多的页面和软件将会被开发出来 (我们已经着手进行其他的项目，致力于简化对VPN、防火墙等的应用)。 这些免费的软件将会使更多的人受益。

1. New documents @ The LDP

  • Lars Strands OLSR IPv6 HOWTO is through with the review procedures. It describes the software and procedures to set up and use the Optimized Link State Routing Protocol with IPv6. The document also contains links to IPv6 pages and a FAQ.

2. Documents submitted for review

  • Rajesh Fowkar submitted updates to the Ext3 mini-HOWTO, correcting some confusing typos.
  • Machtelt Garrels submitted the Bash Beginner's Guide. This document gets you started with Bash scripting and bridges the gap between the Bash HOWTO and the Advanced Bash Scripting Guide. Everybody who wants to make life easier on themselves, power users and sysadmins alike, can benefit from reading this practical course. The guide contains lots of examples and exercises at the end of each chapter, demonstrating the theory and helping you practice.

3. Updated HOWTOs, FAQs and Guides

  • AJ Lewis' LVM HOWTO contains a basic description of Logical Volume Management and explains how it can be implemented on your Linux system. v0.10 Has minor corrections.

4. News in The LDP world

  • LDP review coordinator

    Tabatha Marshall, our review coordinator, takes some time off from TLDP in order to focus on her own publications - she writes novels for teenagers. Meanwhile Emma Jane Hogbin takes over Tabatha's tasks.

The best way to start getting involved and help the LDP would be to first join the <discuss@en.tldp.org> mailing list.

5. HOWTO contribute to The LDP

The Linux Documentation Project (TLDP) is working on developing free, high quality documentation for the GNU/Linux operating system. If you want to help TLDP, consider joining <discuss-subscribe@en.tldp.org>.

Help us create this newsletter. We need volunteers who follow the community (mailing lists, newsgroups, web channels) and report events related to free software documentation. Write to us at <feedback@en.tldp.org>.

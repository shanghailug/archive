四. 主题

  1. Garyshi主讲的GFS(Global File System) 全局文件系统(GFS)是一种能使集群中的服务器直接访问位于同一存储设备上的同一文件系统的技术。由于其能极大降低集群的管理复杂性，并具有优越的性能和扩展性，在企业应用中有着广泛的前景。
    1. 基本概念
      • 系统结构组成
      • 和NFS的区别
      • 性能扩展性
    2. 搭建和使用
      • 软硬件需求
      • 可能的配置方式
      • 软件安装和配置
      • 管理工具和问题解决
    3. 实现技术
      • Red Hat对称集群架构(SCA)
      • 分布式锁管理(DLM)
    4. 其它相关的技术和产品

  2. Posix主讲的 "OpenSource 与企业IT监管“

    介绍使用OpenSource软件实现企业IT管理的几大目标

    1. 资产管理及发掘
    2. 产能管理和预备
    3. 配置及变更管理
    4. 身份认证
    5. 突发事件及问题管理
    6. 基础构件及应用管理
    7. 服务冲突及行为管理
    8. 服务等级管理SLA

    对OpenSource软件与商业软件在企业IT管理领域的各自优劣面进行了比较分析。

  3. Chaos主讲的 “使用Linux进行高性能计算” 介绍Linux的高性能计算集群的搭建，了解构建集群的核心要素和主要方法，初步 了解对集群性能评价的依据，并对影响性能的因素进行简单讨论。
    1. Beowulf, 定义，起源
    2. 软件构成
      1. Linux
      2. MPI Library
      3. 科学计算核心软件库
      4. 资源监控
      5. 性能监控
    3. 硬件构成
      1. 计算结点的内存，CPU
      2. 网络
    4. 性能评价的依据
      1. 并行计算的原理
      2. 影响性能的因素
    5. 对集群性能的检测
      1. 网络带宽和延迟
      2. HPL

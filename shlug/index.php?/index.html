上海Linux用户群介绍

Shanghai Linux User Group (SHLUG) 建立于1997年，是一个在上海地区专注于GNU/Linux及自由/开源软件交流活动的社区。Geekbone是我们的另一个NickName，我们的社区的起源出自一次#unix频道上的闲聊，2000年wuming和chacha首先为这个社区起了这个名字，并贡献了网上空间。shlug是一个以交流技术为主要特点的社区。而诞生它的irc频道#unix则是上海地区老字号的在线技术交流频道 irc.pchome.net. shlug的网站仅仅提供最新的活动消息和debian mirror下载，并不存在交流功能。平日主要的在线交流场所是在irc频道和Google Group的邮件列表. 具体请参见http://groups.google.com/group/shlug)

联系我们最佳方式是： 该E-mail地址已受到防止垃圾邮件机器人的保护，您必须启用浏览器的Java Script才能看到。 / 该E-mail地址已受到防止垃圾邮件机器人的保护，您必须启用浏览器的Java Script才能看到。

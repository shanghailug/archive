在线交流渠道

SHLUG目前提供在线交流方式有邮件列表和IRC实时在线交谈两种方式。

一. 邮件列表 

    （1）订阅方法：
    登录 http://groups.google.com/group/shlug，以自已的gmail帐号登入后加入googlegroup 或者 发送一封含任意主题和内容的电子邮件至 该E-mail地址已受到防止垃圾邮件机器人的保护，您必须启用浏览器的Java Script才能看到。 即可，将subscribe改成unsubscribe则退订。

   （2）该group的范围包括

  1. 与shlug活动相关的讲座主题的征集，活动的时间和地点的通知。
  2. 与geekbone mirror相关的询问和建议。
  3. 与shlug有关的其它各类正式消息的发布。

    （3） 邮件列表的在线阅读和发信地址：http://groups.google.com/group/shlug

二. IRC在线交流

    服务器地址: irc.freenode.net

    交流频道: #shlug

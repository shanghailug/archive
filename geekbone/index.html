GeeK News

     Shanghai Linux User Group ( SHLUG ) 是一个在上海地区专注于Linux交流活动的社区。当前,主要集中于Debian领域的支持。每年年初和年中都会有线下的聚会活动。alias geekbone=SHLUG

     geekbone(shlug)社区的起源出自一次#unix频道上的闲聊，2000年wuming和chacha首先为这个社区起了这个名字，并贡献了网上空间。geekbone这个名字的分别意指geek（极客）和bone（IPV6中的一个常用词根），即技术爱好者群体。所以geekbone是一个以交流技术为主要特点的社区。而诞生它的irc频道#unix则是上海地区老字号的在线技术交流频道irc.pchome.net.geekbone.org这个网站仅仅提供最新的活动消息和debian mirror下载，并不存在交流功能。平日主要的在线交流场所是在irc频道中（#unix和#geekbone）。

     #unix 经过如此的发展后，在一个IRC #unix 频道里形成了一种 UNIX 文化，因此在一个channel内进行大家观点和思想的交流，以及技术方面的讨论，似乎相当不堪重负，因此，#unix在 公元2000年3月，开通了新的频道，并以网站命名 #geekbone。目的在于：#unix将主要用来交流技术，而#geekbone，则期望大家交流各种思想和感情。因此#geekbone 也适用 #unix的大部分聊天规则，如：不得使用bot，不得用彩色字等等。

     tux年志：在2003年的年会之后，由Chaos和Jackz等人提出了建立一套debian mirror的倡议，并得到了大家的支持。不久，Chaos捐出了自已的一套机器作为最初的debian mirror的镜像用机，通过Jackz托管进了外高桥机房。经过大家的这番努力，终于在2003年2月份正式运行了第一代mirrors.geekbone.org。不久,随着下载流量的增大这套服务器已经尽显疲态。为了保障镜像的良好运行，从2003年年底到2004年年初的一个月中，通过#unix里极客们的捐款又重新配置了一台单Xeon 2.4G,1GRAM和4x160G的第二代mirrors.geekbone.org服务器，同时又通过Jackz的努力托进了漕宝路机房。第二代的服务器虽然明显强劲了不少，但是很快就发现经常会发生IO错误，一年的运行期内多次有当机现象发生。2005年1月10日，服务器彻底停摆，后经过Chaos,Jackz,Garyshi等弟兄的检查确定为服务器中所使用的一块2U PCI-X转接卡存在不稳定的问题，换了一块Jackz DIY过的1U转接卡后故障排除。2005年3月30日，服务器在加了一块由scott捐赠的160G硬盘后，再次被托管进了外高桥机房。神佑tux

Debian mirror: http://mirrors.geekbone.org/
